I created this JavaScript/CSS animation to mimick the behavior of the buttons on the Apple TV. The buttons are 3D and have a nice hover effect. I mainly created this as a demo but it could be a nice way to draw users to your CTAs.

In this demo I added 'cursor: none' to the dynamic-button class to make the effect even more convincing.





Usage

1. Import the CSS and JavaScript files 
    
<link rel="stylesheet" href="tvos-style-3d-buttons/css/tvos-style-3d-buttons.css">
<script src="tvos-style-3d-buttons/js/tvos-style-3d-buttons.js"></script>
2. Add the HTML elements using the class "dynamic-button" 
    
<button class="dynamic-button">Button</button>

<div class="dynamic-button">
    Some div content
</div>
