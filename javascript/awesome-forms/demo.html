Awesome Forms

Awesome forms is a super lightweight and easy to implement form validaiton script. It is a passive validation system, meaning it won't bug you with validation errors while you are typing-- only after you click out of a form field or press the submit button will it check for valid entries. Basic usage

1. Import the script

<script src="awesome-forms.js"></script>

Just like that, Awesome Forms is now active on your form!


2. To add an error message/help text to a form field, create a div with the class af-helptext directly after it

<label for="email">Email:</label>
<input type="email" id="email" required>
<div class="af-helptext">Please enter a valid email</div>

These will be automatically hidden on load, and revealed if the script detects an error.




Example usage

Here is the exact code from the example above

<div class="afdemo-form-container">
    <form>
        <div class="afdemo-form-group">
            <label for="name">Name:</label>
            <input type="text" id="name" required>
            <div class="af-helptext">Please enter your name</div>
        </div>

        <div class="afdemo-form-group">
            <label for="tel">Phone:</label>
            <input type="tel" id="tel"required>
            <div class="af-helptext">Please enter a valid 10-digit phone number</div>
        </div>

        <div class="afdemo-form-group">
            <label for="email">Email:</label>
            <input type="email" id="email" required>
            <div class="af-helptext">Please enter a valid email address</div>
        </div>

        <div class="afdemo-form-group">
            <div class="afdemo-h-flex">
                <input type="checkbox" id="consent" checked required>
                <label for="consent">I agree to the terms & conditions</label>
                <div class="af-helptext">Please agree to the terms</div>
            </div>
        </div>

        <div class="afdemo-form-group">
            <textarea style="resize: vertical;"></textarea>
        </div>

        <input type="submit" onsubmit="alert('Form submitted successfully!')">
    </form>
</div>



How it works


Phone number validation

Automatically runs on <input type="tel"> elements. Ensures the input is a ten digit number with no non-numeric characters. When 10 digits are entered, it automatically reformats them to the following format (555) 555-5555.


Email validation

Automatically runs on <input type="email"> elements. Ensures the input is a valid email by making sure it contains an @ symbol with text before and after it, and ends with a domain-- basically anything ending with a dot (".") and something after it.

For example, a@b.c would register as valid but @b.c or a.c would not.


Required field validation

Automatically runs on any element with the required attribute. Ensures the field is not empty, or is not unchecked in the case of checkbox elements.


Mechanism of action

Awesome Forms is a passive validation system, meaning it won't bug you with validation errors while you are typing-- only after you click out of a form field or press the submit button will it check for valid entries.

If the script detects an error, it will block the form from being submitted using event.preventDefault() and will play a subtle animation to emphasize the active help text items.
