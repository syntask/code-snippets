Dynamic Buttons

I designed this JavaScript/CSS hover animation to replicate the 3D animation used on Apple TVs when you select a UI element. It makes buttons or other UI elements feel a bit more physically interactive, and the way they follow the mouse almost feels magnetic.




It looks great on things like images to - just keep in mind for images you need to make a <div> with a background-image property rather than using a regular <img> tag.



You can also create a really cool effect by changing the mouse cursor to a circle and hiding it when UI elements are hovered, similar to using a mouse on iPad OS.




Basic usage

1. Import the CSS and JavaScript files:

<link rel="stylesheet" href="dynamic-buttons.css">
<script src="dynamic-buttons.js"></script>

2. Add the "dynamic-button" class on any element you want to apply the hover animation to:

<button class="dynamic-button">Click me!</button>

That's it! Keep in mind, this will replace the need to implement your own CSS hover states, however you may need to define some additional classes and/or pseudo classes for mousedown or disabled states.
