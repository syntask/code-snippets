By combining the 'user-select: none' and 'cursor: default' CSS properties you can create better UI elements for web apps that the user can't accidentally highlight. For example, try selecting the text below:


This text is not selectable.

This also works great on buttons:



Here is the code:


  
    .noselect {
      user-select: none; /* Standard syntax */
      -webkit-user-select: none; /* Safari syntax*/
      -ms-user-select: none; /* Internet Explorer syntax */
      -moz-user-select: none; /* Firefox syntax */
      cursor: default;
    }
