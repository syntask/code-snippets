I created Awesome Forms as a plug and play form validation script that is highly versatile.

Demo




Usage


1. Include the script in your HTML file: 
    
<script src="awesome-forms.js"></script>

2. Add input fields as you normally would, and add a div with the class "af-helptext" for any error text you want to show. Be sure to include the 'required' attribute on any fields you want to validate. 
    
<form>

    <label for="name">Name:</label>
    <input type="text" id="name" required>
    <div class="af-helptext">Please enter your name</div>

    <label for="tel">Phone:</label>
    <input type="tel" id="tel" required>
    <div class="af-helptext">Please enter a valid 10-digit phone number</div>

    <input type="submit" value="Submit">
    
</form>

Supported field types

    
<!-- Email inputs will be validated automatically -->
<input type="email">

<!-- Tel inputs will be validated and formatted automatically (10 digits) -->
<input type="tel">

<!-- And inputs of any kind which include the 'required' attribute -->
<input type="text" required>
