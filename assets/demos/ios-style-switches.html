I created these switches to mimick the style of iOS's toggle switches. Try them out below:




Basic usage

To use, just import the css file and wrap your checkbox inputs in a label with the class 'toggle-switch' and add an empty 'i' element inside. 
            
<head>

    <!-- Import the css file in your head -->
    <link rel='stylesheet' href='ios-style-switches.css'>

</head>

<body>

    <!-- Add the toggle switches like this -->
    <label class="toggle-switch light">
        <input type="checkbox">
        <i></i>
    </label>

</body>



Customization

To customize the switch colors, you can either use the 'light' and 'dark' classes, or define your own class. 
            
<style>

.toggle-switch.custom-class{
    --track-base-color: #ccc;
    --track-active-color: #34c759;
    --handle-color: #FFF;
    --handle-shadow-color: rgba(0, 0, 0, 0.24);
}

</style>


<!-- Light -->
<label class="toggle-switch light">
    <input type="checkbox">
    <i></i>
</label>

<!-- Dark -->
<label class="toggle-switch dark">
    <input type="checkbox">
    <i></i>
</label>

<!-- Custom -->
<label class="toggle-switch custom-class">
    <input type="checkbox">
    <i></i>
</label>


Sizing

The switches are created using relative units, so they are easily scalable by setting their 'height' property. By default they are 1.5em tall, which allows you to easily insert them inline and have them fit nicely with their label text.






However, you can also set the size explicity by setting the 'height' property of the label to your desired px or em value. Or you can define a new subclass if you like. 
            
.toggle-switch {
    height: 2em; /* Affects all toggle switches on the page */
}

.toggle-switch.small {
    height: 16px; /* Only affects switches with the 'small' class */
}
